# minusGAN
making GAN learn to generate a dataset which is from the dataset X but not from another dataset Y, like the set operation X minus Y.

this code is based on [ALICE](https://github.com/ChunyuanLI/ALICE).

### Dependencies
- `python 2.7 or 3.5`
- `TensorFlow 1.0.0`
